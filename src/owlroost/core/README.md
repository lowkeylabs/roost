The core subsystem contains runtime realization
infrastructure responsible for executing,
monitoring, and enriching ROOST analytical runs.

Responsibilities:
    - OWL execution orchestration
    - runtime progress rendering
    - external HFP ingestion
    - runtime longevity realization
    - execution support helpers

This subsystem intentionally does NOT own:
    - semantic ontology
    - metrics ontology
    - display rendering
    - catalog provenance
